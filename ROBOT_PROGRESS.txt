PROJECT GOAL
═══════════════════════════════════════════════════════════════
UNIT_01 — Autonomous Indoor Navigation Robot

An autonomous mobile robot designed for industrial indoor
environments (hospitals, warehouses, offices, hotels) that:

- Navigates autonomously using a LiDAR-built map (SLAM once,
 navigate forever)
- Uses AprilTag checkpoints as instruction triggers
 (stop, go to goal, turn, follow, etc.)
- Avoids obstacles in real time using LiDAR
- Streams live camera feed via WebRTC to operator dashboard
- Controlled and monitored via a web dashboard accessible
 from any browser on the same network (later: internet)
- Supports multi-level override: AUTO / SUPERVISED / MANUAL /
 EMERGENCY STOP — with collision guard always active
- Animated Echo face HUD on built-in 4.3" touchscreen showing
 robot state and expressions
- Voice interaction: 24/7 background listener for wake word "Echo"
 and high-quality TTS voice for personality-driven responses
- Powered by OpenClaw + Claude Opus 4.6 as AI brain:
 * Robot explores autonomously with no user commands
 * Discovers and names new areas via 360° camera scan
 * Dynamically adds goto_<name> commands to locations.json
 * SOUL.md personality: curious, self-directed explorer
 * Telegram = observation interface (watch, don't control)
- Logs all missions with timestamps (checkpoint history)
- Login-protected web dashboard with Firebase Auth
- Fully offline capable for navigation and safety
 (AI brain requires Anthropic API)

CHASSIS:
- Box shape, MDF bottom, plexiglass sides, laser cut finger
 joints, Freenove case as front face
- 4 mecanum wheels (80mm) for omnidirectional movement
- All electronics inside the box
- Motors mounted on MDF bottom (outside)

FUTURE EXPANSION:
- Robot arm for picking/placing objects
- Thermal + gas sensors for anomaly detection
- Multi-robot fleet management
- Internet access via TURN server
- Face recognition for access control
